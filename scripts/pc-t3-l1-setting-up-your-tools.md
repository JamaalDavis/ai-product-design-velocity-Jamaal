# Script: Setting Up Your Tools
**Module:** Pre-Course — Topic 3, Lesson 1
**Instructor:** Simon
**Duration:** 5-6 min
**Format:** Talking head + screen recording

---

## LESSON GOAL
Walk students through the three installs they need before anything else: a GitHub account, VS Code, and the Claude Code extension. Pure setup — mechanical, sequential, skip-able for students who already have these. Ends with a clear "you're done" state and handoff to the next lesson.

## RECORDING NOTES
- Start talking head for 20 seconds, then go to screen for the full sequence
- Record on a clean machine or hide personal files
- Light mode throughout
- Keep the pace moving — this is install steps, not teaching. Every step ends with a clear "you're done when you see X"
- Don't explain what these tools are in depth — that comes in the environment tour (L2)

---

## SCRIPT

### [0:00–0:20] INTRO

**[Face camera]**

"Before we do anything else, you need three things installed: a GitHub account, VS Code, and the Claude Code extension.

This lesson is just the installs. It'll take about five minutes."

---

### [0:20–1:30] GITHUB ACCOUNT

**[Screen share: browser → github.com]**

"First: GitHub. Go to github.com.

If you already have an account, log in and move on. If not, click Sign Up and create a free account — just an email address.

**[Show signup flow briefly]**

GitHub is where the course files live. You'll pull them onto your machine in the next lesson. You don't need to understand Git deeply — you just need the account.

You're done when you see: your GitHub dashboard."

---

### [1:30–3:00] INSTALL VS CODE

**[Screen share: browser → code.visualstudio.com]**

"Now VS Code. Go to code.visualstudio.com.

**[Point to the download button]**

Click the download button for your operating system — it usually detects yours automatically.

**[Download starts]**

Once it downloads, open the installer. On Mac, drag VS Code into your Applications folder. On Windows, run the .exe and accept the defaults.

**[Open VS Code — welcome screen visible]**

Open it. You'll see the welcome screen.

You're done when you see: VS Code open on your screen."

---

### [3:00–4:30] INSTALL CLAUDE CODE EXTENSION

**[Screen share: VS Code open — click the Extensions icon in the left sidebar]**

"Now Claude Code. This is an extension that lives inside VS Code — you don't install it separately.

Click the Extensions icon in the left sidebar — it looks like four squares.

**[Type 'Claude Code' in the search bar]**

Search for Claude Code. You want the one published by Anthropic.

**[Click Install]**

Click Install. Takes about 10 seconds.

**[Claude Code icon appears in sidebar]**

When it's done, you'll see the Claude Code icon in your sidebar.

You're done when you see: the Claude Code icon in your VS Code sidebar."

---

### [4:30–5:00] CLOSE

**[Face camera]**

"That's all three. GitHub account, VS Code, Claude Code extension.

In the next lesson, I'll show you how to get the course files onto your machine and walk you through the working environment. See you there."

**[End]**
